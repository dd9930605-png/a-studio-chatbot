## 上傳注意

1. 若上傳 `M5.png`，請確認 `outfits.json` 寫的是 `/images/outfits/M5.png`（不是 `.jpg`）
2. 建議與 Wix 網站使用相同原圖
3. 上傳後可用無痕模式開啟 `/pre` 檢查 12 套是否更新

舊的 `.jpg` 檔若未刪除不影響網站，只要 `outfits.json` 指向新檔即可。
