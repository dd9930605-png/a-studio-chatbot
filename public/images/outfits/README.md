## 更換圖片

1. 將新圖覆蓋 `public/images/outfits/M3.jpg`（等 5 個檔案）
2. 建議使用與 Wix 網站相同的原圖，確保前台與 AI 系統一致
3. 格式：JPG，直式商品圖，白底為佳

文案由 `public/outfits.json` 的 `reason` / `benefit` / `limitation` / `suggestion` 自動產生，無需另改程式。
